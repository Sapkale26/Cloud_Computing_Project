# Database Setup

## Overview

This project uses PostgreSQL 18 as the database for storing event information generated by the Edge AI Monitoring System.

## Database Information

* Database Name: edge_monitor
* Database User: postgres
* Host: localhost
* Port: 5432

## PostgreSQL Installation

PostgreSQL 18 was installed locally on macOS using the EnterpriseDB installer.

## Database Creation

```sql
CREATE DATABASE edge_monitor;
```

## Events Table

```sql
CREATE TABLE events (
    id SERIAL PRIMARY KEY,
    event_type VARCHAR(100),
    location VARCHAR(100),
    image_url TEXT,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);
```

## Verification

The database connection was verified using:

```bash
psql -U postgres
```

and

```sql
\c edge_monitor
\dt
```

## Sample Insert

```sql
INSERT INTO events(event_type, location, image_url)
VALUES ('Fire', 'Building A', 'fire.jpg');
```

## Result

The PostgreSQL database successfully stores and retrieves event data for the backend API.
