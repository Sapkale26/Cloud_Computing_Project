# Backend Setup Guide

## Step 1

Install Node.js:
https://nodejs.org

---

## Step 2

Initialize project:

```bash
npm init -y
```

---

## Step 3

Install Express:

```bash
npm install express
```

---

## Step 4

Create server.js

---

## Step 5

Run server:

```bash
node server.js
```

---

## Step 6

Open browser:

```text
http://localhost:3000
```

---

## Step 7

Test APIs using Postman.
